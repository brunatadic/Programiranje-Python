Python programski jezik

python je programski jezik opće namjene, interpreterski jezik



python programerima dopušta nekoliko stilova pisanja programa

  • strukturno
  • objektno orijentirano
  • aspektno orijentirano


Python se koristi u gotovo svakoj industriji i znanstvenom području koje možete zamisliti, uključujući:

  • Znanost o podacima
  • Strojno učenje
  • Web razvoj
  • obrazovanje iz informatike
  • Računalni vid i obrada slika
  • Razvoj igara
  • Medicina i farmakologija
  • Biologija i bioinformatika
  • Neuroznanost i psihologija
  • Astronomija
  • Ostala područja poput robotike, autonomnih vozila, poslovanja, meteorologije i razvoja grafičkog korisničkog sučelja (GUI)

Razvoj Pythona u velikoj se mjeri odvija kroz postupak prijedloga poboljšanja Pythona (PEP), primarnog mehanizma za predlaganje velikih novih značajki, prikupljanje doprinosa zajednice o pitanjima i dokumentiranje odluka o dizajnu Pythona. Python stil kodiranja pokriven je PEP -om 8. Izvanredne PEP -ove pregledavaju i komentiraju zajednica Python i upravno vijeće.



osnovni tipovi podataka u pythonu su:

  • cjelobrojni (interger) x=10
  • decimalni (float) x=3.14
  • kompleksni brojevi (complex) x=1j
  • tekstualni (string) x="Dobar dan"
  • logički (Booleov tip podataka) x=True ili x=False
  • none-varijable bez postavljene vrijednosti
