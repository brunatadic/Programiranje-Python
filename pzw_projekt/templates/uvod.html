uvod u programiranje

programiranje

Programiranje je pisanje uputa računalu što i kako učiniti, a izvodi se u nekom od programskih jezika. Stvaranje programa sadrži u sebi pojedine elemente dizajna, umjetnosti, znanosti, matematike kao i inženjeringa. Osoba koja stvara program zove se programer.



programski jezici

Programi ili upute za računalo pišu se u programskom jeziku upotrebom određene sintakse i pravila koja vrijede za svaki programski jezik (ili tip), koji se potom prevodi u strojni jezik koje je osobito za određeno računalo te je ovisno o njegovoj arhitekturi.

  • Assembler
  • BASIC
  • Pascal
  • C/C++/C#
  • Java
  • Logo
  • Fortran
  • Python

Faze razvoja programiranja

pri izradi svakog programa potrebno je proći kroz 6. osnovne faze ili koraka:

  1. shatiti problem koji je potrebno riješiti
  2. rastaviti problem na manje dijelove (module)
  3. prepoznati programski elementi potrebni za rješavanje problema
  4. povezati programske elemente u smislen algoritam
  5. napisati program u odabranom programskom jeziku
  6. testirati program, pronaći rubne slučajeve te popraviti eventualno greške



program

program je niz naredbi koje se izvode točno određenim redoslijedom s točno određenim ciljem. Naredba je temeljni dio programa. Programe izrađuju programeri koristeći jedan od programskih jezika. Stvaraju naredbe za aktivnosti računala koje se pokreću na temelju korisnikovih akcija (pritiska miša, tipkovnice ili ekrana). Korisnik komunicira s računalom s pomoću programa.
